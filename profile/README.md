# SoftVoyagers

**Free developer tools & APIs** built in Poland, available worldwide.

We build and maintain a growing portfolio of free, no-signup-required APIs and developer tools. Every product is open for public use — no API keys, no credit cards, no limits on getting started.

## Our Products

| Product | Description | Website |
|---------|-------------|---------|
| **LinkMeta** | URL metadata extraction API — get titles, descriptions, images & Open Graph data from any URL | [linkmeta.dev](https://linkmeta.dev) |
| **PageShot** | Screenshot & webpage capture API — render any URL as PNG | [pageshot.site](https://pageshot.site) |
| **PDFSpark** | HTML/URL to PDF conversion API — generate PDFs from HTML or web pages | [pdfspark.dev](https://pdfspark.dev) |
| **OGForge** | Open Graph image generator API — create OG images with 4 themes and 1,668 icons | [ogforge.dev](https://www.ogforge.dev) |
| **LinkShrink** | Privacy-first URL shortener — no tracking, no ads | [linkshrink.dev](https://linkshrink.dev) |
| **MyAddress.ai** | Address autocomplete & geocoding engine | [myaddress.ai](https://myaddress.ai) |
| **Faktuj** | Darmowy generator faktur VAT online (Polish invoice generator) | [faktuj.pl](https://faktuj.pl) |

## Tech Stack

Our APIs are built with **Node.js**, **TypeScript**, and **Express**, deployed on **Azure** via **Docker** containers with **GitHub Actions** CI/CD.
